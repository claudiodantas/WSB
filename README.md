# WEBService Biblioteca

## Descrição:
Este é um WebService para a utilização de recursos de uma biblioteca, visando facilitar a transação de processos entre os usuários da biblioteca e seu sistema de empréstimos, renovação de livros, pagamentos de multas e verificação do acervo, entre outros.
